4  Die ökonomische Analyse sozialer Phänomene

4.1 Soziale Phänomene und individuelles Handeln

  • Soziale Phänomene entstehen durch Entscheidungen und Interaktion von Individuen.

  • Sie sind regelmäßig „Ergebnis menschlichen Handelns aber nicht menschlichen Entwurfs“ (Hayek) (spontane Ordnung)

4.2 Gleichgewichte

  • Ein Gleichgewicht liegt vor, wenn ein System ohne „Störungen“ durch äußere Einflüsse in Ruhe verbleibt.
  • Muster: Individuelle Entscheidungen \(\rightarrow\) Interaktion/Anpassung/Aggregation \(\rightarrow\) Gleichgewicht. Exogener Schock \(\rightarrow\) neue individuelle Entscheidungen \(\rightarrow\) Interaktion/Anpassung/Aggregation \(\rightarrow \dots\)
Sind wir je im Gleichgewicht?

Einleitung

In ökonomischen Analysen spielt der Gleichgewichtsbegriff eine zentrale Rolle.

Ein (ökonomisches) Gleichgewicht ist ein (Modell-)Zustand, der stabil ist, wenn keine zusätzlichen Einflüsse auf ihn wirken.

Folgende gedachte Abfolge ist in ökonomischen Modellen üblich:

  • Ein System befindet sich in einem Gleichgewicht

  • Ein exogener Schock wirkt auf das System und bewegt ein Modellelement aus dem Gleichgewicht heraus und/oder verändert die Bedingungen für das Vorliegen eines Gleichgewichts

  • Die Modellakteure passen ihre Entscheidungen an, sodass wieder ein Gleichgewicht erreicht wird.

Der Übergang in das Gleichgewicht wird in der Regel nicht explizit analysiert, sondern es werden nur gleichgewichtige Zustände (und ggf ein ungleichgewichtiger Zustand nach dem Schock) miteinander verglichen (komparative Statik).

Gegebenenfalls kann aber geprüft werden, ob das System zu einem Gleichgewicht strebt oder nicht.

Referenzszenario: Die Welt im Gleichgewicht

Stellen wir uns zunächst als Referenzfall eine Welt vor, die nur aus Gleichgewichtszuständen besteht. Hierzu nehmen wir an, dass wir Ausprägungen einer Variable \(y\) zu unterschiedlichen Zeitpunkten \(t\) beobachten können. Nehmen wir an, dass wir eine theoretische Vorstellung über die Entwicklung von \(y\) haben, sodass gilt: \[y_t=\underbrace{e^{rt}}_{\text{Gleichgewicht zum Zeitpunkt } t}\]

\(y\) wächst also in \(t\) mit einer Rate von \(r~\%\). Grafisch lässt sich das wie folgt veranschaulichen. Jede Beobachtung (schwarze Punkte) liegt exakt auf der blauen Linie des gleichgewichtigen Wachstumspfads.

In jeder Periode ist das Gleichgewicht auch der erwartete und der tatsächliche Wert.

Code 
library(tidyverse)

set.seed(1234)

r <- .013 # Trend
beta <- .6 # Faktor für Gedächtnis
sigma <- .1 # Varianz

df <- data.frame(
       Zeit   = c(0:49),
       Schock = c(rnorm(30, 0, sigma), rep(0,20))
) %>% mutate(
   Trend = exp(Zeit*r),
   Gedächtnis = accumulate(Schock, ~ beta * .x + .y))
# Trend

df %>% ggplot(aes(Zeit, Trend)) +
   geom_line(color='blue') +
   geom_point() +
   theme_light()

Die Realität außerhalb des Gleichgewichts

Die analytische Fokussierung auf Gleichgewichtszustände bedeutet nicht, dass sich ein gleichgewichtiger Zustand ohne weiteres beobachten lässt. Im Gegenteil, man geht in empirischen Analysen regelmäßig davon aus, dass zu jedem Zeitpunkt eine exogene Störung auf den zu beobachtenen Sachverhalt wirkt. Diese Störung, oder dieser Schock, entfernt das System aus dem gleichgewichtigen Zustand. Typischerweise wird dabei unterstellt, dass die Störungen unabhängig voneinander normalverteilt sind.

Eine Möglichkeit sich diese Störungen vorzustellen besteht darin, dass sie zu jedem Beobachtungszeutpunkt eine Störung auf auf das System wirkt, das sich im Gleichgewicht befindet.

\[y_t = \underbrace{e^{rt}}_{\text{Gleichgewicht zum Zeitpunkt } t} + \underbrace{\epsilon_t}_{\text{Schock}}\]

Jede einzelne Beobachtung zum Zeitpunkt \(t\) setzt sich also zusammen aus dem gleichgewichtigen Trend (dem durch ein Modell erklärten Teil einer Beobachtung), und einem jeweils einmaligen zufälligen Schock. Die Schocks wirken nur temporär, sie haben keine Auswirkungen auf zukünftige Beobachtungen. Ein Modellgleichgewicht kann also nie die ganze Realität erklären. Es bleibt immer ein unerklärter Rest.

Sobald keine neuen Schocks hinzukommen (hier ab Periode \(t=30\)) liegen alle Beobachtungen auf dem Gleichgewichtspfad. Das Gleichgewicht ist also der Systemzustand in Abwesenheit gegenwärtiger Schocks.

In jeder Periode ist das Gleichgewicht zwar nicht der tatsächliche aber der erwartete Wert.

Code 
# Trend u Schock

df %>%
 ggplot(aes(Zeit, Trend + Schock)) +
   geom_line() +
   geom_point() +
 #  geom_smooth(aes(x=ifelse(Zeit < 31, Zeit, NA)),method= 'glm',
 #              formula = y ~ x + I(x^2)) +
   geom_line(aes(y= Trend), color= 'blue') +
   geom_vline(xintercept = 30, linetype ='dashed') +
   theme_light()

Trend und persistente Schocks

Die Vorstellung, dass jeder Schock noch in der laufenden Periode absorbiert wird und jede neue Periode ihren Schock aus dem Gleichgewicht heraus erhält, ist vielleicht nicht immer ganz passend. Man kann sich auch vorstellen, dass ein Schock über mehrere Perioden abgebaut wird und in der Zeit neue Schocks auf das System einwirken. Eine Beobachtung zum Zeitpunkt \(t\) ist dann das Resultat aus dem eigentlichen Gleichgewicht, einem aktuellen Schock in Periode \(t\) und der Summe der (gewichteten) vergangenen Schocks. Hierbei wird das Gewicht der Schocks immer kleiner, je länger sie vergangen sind. Dies lässt sich wie folgt notieren:

\[y_t= \underbrace{e^{rt}}_{\text{Gleichgewicht zum Zeitpunkt }t} + \underbrace {\sum_{\tau=0}^t \beta^{t - \tau} \epsilon_\tau}_{\text {Schockgedächtnis}}\]

Jede Störung, die sich zum Zeitpunkt \(\tau\) ereignet hat, wird zum Zeitpunkt \(t\) mit \(\beta^{t-\tau}\) gewichtet, mit \(0 \leq \beta \leq 1\). \(\tau\) bezeichnet die Periode, in der die Störung auf das Systen wirkt. Schocks aus entfernteren Perioden erhalten ein geringeres Gewicht als aktuelle Schocks. Interessant sind zwei Grenzfälle: \(\beta = 0\) ist gleichbedeutend mit dem Fall aus dem vorherigen Abschnitt, in dem die Schocks nur in der aktuellen Periode wirken (man beachte für die aktuelle Periode, dass \(\beta^{t - \tau} = 0^0 = 1\)). \(\beta = 1\) beschreibt den Fall, dass alle Schocks unvermindert auf die Gegenwart wirken. In dem Fall würden Beobachtungen auch ohne das Hinzukommen neuer Schocks außerhalb des Trends verharren (die Schocks hätten einen neuen gleichgewichtigen Pfad bewirkt).

Die folgende Abbildung zeigt eine Folge von Beobachtungen für \(\beta = 0,6\). Man sieht, dass die Beobachtungen anders um den Trend angeordnet sind als in der Abbildung davor. Sobald keine neuen Störungen hinzukommen, konvergieren die Beobachtungen wieder gegen den Trend. Das Gleichgewicht beschreibt also einen Zustand, auf den sich ein System in der Abwesenheit neuer Schocks zubewegt.

 ggplot(aes(Zeit, Trend + Gedächtnis)) +
   geom_line() +

  #              formula = y ~ x + I(x^2)

4.3 Erklärungen durch Modelle

(wissenschaftliche) Erklärung: Antwort auf eine Warum-Frage unter Rückgriff auf systematische Zusammenhänge

Theorie: Systematische Vorstellung über Aspekte der Welt

Modell: Theoriegeleitetes Abbild einer Situation

Modellgestützte Erklärung:

  • Systematischer Zusammenhang zwischen unabhängigen und abhängigen Variablen.

  • Erlaubt die Formulierung einer Erwartung über Entwicklung der abhängigen Variable in Abhängigkeit von der Entwicklung der unabhängigen Variable(n).

  • Komplexität des Modells steuerbar
    \(\rightarrow\) möglichst einfache Modelle verwenden

  • Fokussierung auf interessante Zusammenhänge

  • Erlaubt keine Prognose von Einzelereignissen

  • In jedem Einzelfall wirkt eine Vielzahl von unsystematischen Einzelfaktoren, die sich nicht systematisch erklären lassen.

Code 
library(tidyverse)

set.seed(77)

df <- data.frame(x = 0:100) %>%
  mutate(y = exp(x)^.02,
         obs = y + rnorm(n = n(), mean = 0, sd = .5))

model <- df %>% lm(obs ~ x + I(x^2), .)
df$fit <- fitted(model)

ggplot(df, aes(x, obs)) +
  geom_point(aes(color = "Beobachtungen")) +
  geom_line(aes(y = fit, color = "Erklärter Zusammenhang")) +
  geom_segment(aes(x = x, xend = x, y = obs, yend = fit, color = "Unerklärter Rest"), alpha = 0.5) +
  scale_color_manual(values = c("Beobachtungen" = "black", "Erklärter Zusammenhang" = "blue", "Unerklärter Rest" = "red")) +
  scale_x_continuous(breaks=NULL)+
  scale_y_continuous(breaks=NULL)+
  labs(title= "Erklärter und unerklärter Teil von Bebachtungen",
      x= "Unabhängige Variable",
      y= "Abhängige Variable",
      color = "",
      caption="Darstellung: Jan S. Voßwinkel") +
  theme_light()+
  theme(legend.position = "bottom")
  • Modelle erlauben es, den Grad an Komplexität zu steuern.
    • Beschränkung (zunächst) auf Zusammenhänge, von denen man annimmt, dass sie stark sind
    • dann ggf sukzessive Aufnahme weiterer Zusammenhänge

4.4 Typische Grundannahmen in ökonomischen Modellen

  • Methodologischer Individualismus „[A]lle sozialen Phänomene, insbesondere das Funktionieren der sozialen Institutionen [sollten] immer als das Resultat der Entscheidungen, Handlungen, Einstellungen menschlicher Individuen verstanden werden und (…) wir [dürfen] nie mit einer Erklärung aufgrund sogenannter ‘Kollektive’ (Staaten, Nationen, Rassen usw.) zufrieden sein.“ (Karl Popper 1992, S. 116)

  • Normativer Individualismus Die Ziele keines einzigen Menschen sind a priori wichtiger als die irgendeines anderen Menschen. Ausschließlich Individuen und alle Individuen sind Quelle von Werten.

  • Rationalität der handelnden Subjekte wird unterstellt (Verknüpfung von Situation und Handlung möglich)

  • Präferenzen werden als gegeben angenommen

  • Veränderliche Restriktionen bestimmen folglich Ergebnisse
